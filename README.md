# pepprime

Static Astro website with responsive design for desktop and phone.

## Run locally

```bash
npm install
npm run dev
```

Then open the local address shown in the terminal, usually:

```text
http://localhost:4321
```

## Where to edit

- `src/data/products.ts` -> products and product detail pages
- `src/pages/about.astro` -> About Us page
- `src/pages/contact.astro` -> contact information and inquiry section
- `public/images/` -> logo and product images

## Notes

- No database is used.
- The site is responsive for PC, Android, and iPhone browsers.
- Contact flow is inquiry only.
